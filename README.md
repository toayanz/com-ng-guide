# com-ng-guide

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库本身不针对任何特定域名或具体网站，仅作为静态 HTML 文件的存放与展示空间。

## 项目简介

本仓库主要收集和维护一系列独立编写的 HTML 页面。这些页面主题分散，每个页面均为独立的静态文件，互不依赖。仓库的目的在于提供一个整洁、可访问的归档环境，方便浏览和引用。

## 目录说明

- `index.html` — 仓库首页，展示页面列表或导航信息
- `pages/` — 存放主要 HTML 页面文件的目录
- `assets/` — 存放页面使用的公共资源（如样式表、图片、脚本等）
- `README.md` — 即本文件，仓库说明

每个独立页面通常以 `*.html` 形式直接放置在 `pages/` 目录下，便于直接访问或嵌入链接。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务
- 页面之间相互独立，不共享状态或数据
- 部分页面可能包含简单的 CSS 和 JavaScript，用于改善展示效果
- 页面内容不涉及特定网站、域名或商业推广
- 归档内容可能不定期更新或移除

## 维护说明

- 欢迎提交 issue 反馈页面问题或建议
- 如需添加新页面，请遵循现有目录结构和命名风格
- 提交 Pull Request 前请确保页面在主流浏览器中正常工作
- 本仓库不欢迎包含恶意代码、跟踪器或自动跳转的页面
